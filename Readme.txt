How to run the online Birth Certificate System Project

1.Download the zip file

2.Extract the file and copy obcs folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5.Create a database with name obcsdb

6.Import obcsdb.sql file

7.Run the script http://localhost/obcs

8.Click on user tab and create ur account.

9. Then login with contact number and pwd.

10. Then go to add details and fill that form log out.

11. Go to admin tab

     also u can export data to excel.

13. again log in to ur own account and go to Certificate and click view and download ur certificate.

In this u can apply for online birth certificate and get it downloaded.
